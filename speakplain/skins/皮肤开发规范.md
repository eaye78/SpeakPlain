# SpeakPlain 皮肤开发规范

## 概述

SpeakPlain 支持通过 JSON 配置文件自定义悬浮框皮肤。自定义皮肤会自动加载到系统设置的皮肤选项中。

## 文件位置

自定义皮肤 JSON 文件可以放在以下位置：
- 应用数据目录下的 `skins/` 文件夹
- 通过设置界面上传加载

## JSON 结构

```json
{
  "id": "unique-skin-id",
  "name": "皮肤名称",
  "description": "皮肤描述",
  "author": "作者名",
  "version": "1.0.0",
  "colors": {
    "background": "#e8ecf3",
    "backgroundGradient": "linear-gradient(135deg, #667eea 0%, #764ba2 100%)",
    "textPrimary": "#4a5568",
    "textSecondary": "#8a9ab8",
    "textActive": "#2563eb",
    "waveformPrimary": "#3b6beb",
    "waveformSecondary": "#667eea",
    "waveformGradient": "linear-gradient(90deg, #3b6beb 0%, #667eea 100%)",
    "dragDot": "#b0bcd0",
    "dragDotHover": "#2563eb",
    "processingDot": "#3b6beb",
    "shadowLight": "#ffffff",
    "shadowDark": "#c5cad4"
  },
  "dimensions": {
    "borderRadius": 12,
    "paddingX": 14,
    "paddingY": 0,
    "gap": 8
  },
  "animations": {
    "transitionDuration": "0.3s"
  }
}
```

## 字段说明

### 必填字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | string | 唯一标识符，只能包含字母、数字、连字符 |
| `name` | string | 皮肤显示名称 |
| `colors.background` | string | 背景色（HEX格式） |
| `colors.textPrimary` | string | 主文字颜色 |
| `colors.waveformPrimary` | string | 波形主色 |

### 可选字段

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `description` | string | - | 皮肤描述 |
| `author` | string | - | 作者名 |
| `version` | string | - | 版本号 |
| `colors.backgroundGradient` | string | background值 | 渐变背景，支持CSS渐变语法 |
| `colors.textSecondary` | string | #8a9ab8 | 次要文字颜色 |
| `colors.textActive` | string | #2563eb | 激活状态文字颜色 |
| `colors.waveformSecondary` | string | waveformPrimary | 波形次要色 |
| `colors.waveformGradient` | string | waveformPrimary | 波形渐变 |
| `colors.dragDot` | string | #b0bcd0 | 拖拽点颜色 |
| `colors.dragDotHover` | string | dragDot | 拖拽点悬停颜色 |
| `colors.processingDot` | string | #3b6beb | 处理中动画点颜色 |
| `colors.shadowLight` | string | #ffffff | 新拟态亮部阴影 |
| `colors.shadowDark` | string | #c5cad4 | 新拟态暗部阴影 |
| `dimensions.borderRadius` | number | 12 | 圆角半径（px） |
| `dimensions.paddingX` | number | 14 | 水平内边距（px） |
| `dimensions.paddingY` | number | 0 | 垂直内边距（px） |
| `dimensions.gap` | number | 8 | 元素间距（px） |
| `animations.transitionDuration` | string | "0.3s" | 过渡动画时长 |

## 颜色格式

支持以下颜色格式：
- HEX: `#ff0000`, `#f00`
- RGB: `rgb(255, 0, 0)`
- RGBA: `rgba(255, 0, 0, 0.5)`
- CSS渐变: `linear-gradient(135deg, #667eea 0%, #764ba2 100%)`

## 内置皮肤参考

### 经典（classic）
```json
{
  "colors": {
    "background": "#e8ecf3",
    "textPrimary": "#4a5568",
    "textSecondary": "#8a9ab8",
    "textActive": "#2563eb",
    "waveformPrimary": "#3b6beb"
  }
}
```

### 彩虹（rainbow）
```json
{
  "colors": {
    "background": "#e8ecf3",
    "backgroundGradient": "linear-gradient(135deg, #667eea 0%, #764ba2 100%)",
    "textActive": "#667eea",
    "waveformPrimary": "#667eea",
    "waveformGradient": "linear-gradient(90deg, #667eea 0%, #764ba2 100%)"
  }
}
```

### 沙漠（desert）
```json
{
  "colors": {
    "background": "#f5e6d3",
    "backgroundGradient": "linear-gradient(135deg, #d4a574 0%, #c17f4e 100%)",
    "textActive": "#c17f4e",
    "waveformPrimary": "#c17f4e"
  }
}
```

### 星空（starry）
```json
{
  "colors": {
    "background": "#1a1f2e",
    "backgroundGradient": "linear-gradient(135deg, #1a1f2e 0%, #2d3748 50%, #1a202c 100%)",
    "textPrimary": "#e2e8f0",
    "textActive": "#63b3ed",
    "waveformPrimary": "#63b3ed"
  }
}
```

## 设计建议

1. **对比度**：确保文字颜色与背景色有足够对比度，保证可读性
2. **一致性**：保持颜色风格一致，避免过于杂乱的配色
3. **可读性**：激活状态颜色应与主色调协调
4. **新拟态**：如需新拟态效果，请同时设置 `shadowLight` 和 `shadowDark`

## 验证

上传皮肤文件前，可以使用在线 JSON 验证工具检查格式是否正确。
